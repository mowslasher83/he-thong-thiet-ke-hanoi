# Hệ Thiết Kế Hà Nội

## Giới thiệu
Hệ thống thiết kế Hà Nội là một bộ công cụ toàn diện nhằm nâng cao quy trình phát triển và thiết kế sản phẩm số tại Việt Nam. Dự án này hướng đến việc tạo ra các thành phần giao diện người dùng có thể tái sử dụng, cùng với các nguyên tắc thiết kế mở rộng thông qua tài liệu và ví dụ điển hình.

## Tính năng
- Thư viện các thành phần UI đã được thiết kế sẵn.
- Tài liệu hướng dẫn sử dụng và tối ưu hóa từng thành phần.
- Ví dụ thực tế cho từng thành phần để người dùng dễ dàng áp dụng.
- Công cụ tạo bộ mã màu và kiểu chữ cho thương hiệu.

## Cài đặt
Để sử dụng hệ thống thiết kế này,

```bash
# Clone the repo
git clone https://github.com/username/he-thong-thiet-ke-hanoi.git

# Cài đặt các package cần thiết
npm install
```

## Sử dụng
Tham khảo tài liệu trong thư mục `docs` để tìm hiểu cách sử dụng từng thành phần.

## Đóng góp
